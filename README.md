# Сайт для воспоминаний
PlacesRemember - место, где вы можете делиться своими воспоминаниями и отзывами про разные места по всему миру!


## Технологии
Python 3.11, Django 4, Django allauth, Vk API, HTML, CSS, Bootstrap, SQLite
